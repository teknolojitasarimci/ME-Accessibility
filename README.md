# ME-Accessibility
Web sitelerini herkes için kapsayıcı hale getiren, %100 Vanilla JS ile yazılmış, bağımlılıksız ve WCAG 2.1 uyumlu erişilebilirlik katmanı. Sesli Okuma (TTS), Disleksi Modu, Yüksek Kontrast ve Akıllı Hafıza özellikleriyle modern bir çözüm. // A lightweight, zero-dependency, WCAG 2.1 compliant accessibility widget with Glassmorphism UI.
